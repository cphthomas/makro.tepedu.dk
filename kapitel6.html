Varemarkedet på kort sigt

Dette kapitel handler om varemarkedet på kort sigt - hvordan efterspørgsel og udbud fungerer på kort sigt og hvordan økonomiske udsving opstår. Som finansøkonom skal du forstå AD-AS modeller, hvordan efterspørgslen påvirkes af forskellige faktorer, og hvordan markedet finder ligevægt.

Hvad er varemarkedet?

Varemarkedet er markedet for varer og tjenester. Det bestemmes af efterspørgsel (hvad folk vil købe) og udbud (hvad virksomheder vil producere).

1. Efterspørgslen - Hvad bestemmer, hvor meget folk vil købe?

Efterspørgslen efter varer og tjenester afhænger af flere faktorer:

Faktor Effekt Eksempel
Indkomst Højere indkomst → Mere forbrug Hvis din løn stiger, køber du typisk flere varer
Priser Laveste priser → Mere forbrug Hvis prisen på biler falder, køber flere biler
Renter Lav rente → Mere forbrug Hvis renten falder, låner folk mere og køber flere varer
Forventninger Optimisme → Mere forbrug Hvis folk tror, økonomien bliver bedre, køber de mere nu
Offentlige udgifter Højere offentlige udgifter → Mere efterspørgsel Hvis staten bygger flere veje, stiger efterspørgslen efter beton og asfalt

2. Udbuddet på kort sigt - Hvordan reagerer producenterne?

På kort sigt er produktionskapaciteten ofte fast, så udbuddet reagerer primært på priserne:

  • Højere priser: Producenterne vil producere mere, fordi de kan tjene mere
  • Lavere priser: Producenterne vil producere mindre, fordi de tjener mindre
  • Produktionsomkostninger: Hvis omkostningerne stiger, vil producenterne producere mindre til samme pris

Vigtigt: På kort sigt kan producenterne ikke ændre produktionskapaciteten meget. De kan kun producere mere eller mindre med de eksisterende fabrikker og maskiner.

Samlet udbud		 Den samlede produktion af varer og tjenester i hele økonomien Priser, produktionsomkostninger, teknologi, produktionskapacitet

4. Ligevægt på varemarkedet

Ligevægt opstår, når efterspørgsel og udbud er lige store:

  • Hvis efterspørgslen er større end udbuddet: Priserne stiger, og producenterne producerer mere
  • Hvis udbuddet er større end efterspørgslen: Priserne falder, og producenterne producerer mindre
  • I ligevægt: Efterspørgsel = Udbud, og priserne er stabile

5. Økonomiske udsving - Hvordan opstår konjunktursvingninger?

Økonomiske udsving opstår, når efterspørgslen eller udbuddet ændrer sig:

Type udsving Hvad sker der? Effekt
Efterspørgselschok Efterspørgslen ændrer sig pludseligt Hvis efterspørgslen stiger, stiger produktionen og priserne
Udbudschok Udbuddet ændrer sig pludseligt Hvis udbuddet falder (fx pga. højere oliepriser), falder produktionen, men priserne stiger

Eksempel på efterspørgselschok: Hvis staten pludseligt øger udgifterne markant, stiger efterspørgslen. Producenterne producerer mere, og priserne stiger.

Eksempel på udbudschok: Hvis olieprisen stiger markant, stiger produktionsomkostningerne. Producenterne producerer mindre, og priserne stiger.

6. Multiplikatoreffekten - Hvordan spreder ændringer sig?

Multiplikatoreffekten betyder, at en ændring i efterspørgslen kan have større effekt på produktionen:

Eksempel:

  1. Staten øger udgifterne med 100 mia. kr. (bygger veje)
  2. Vejbyggerne får løn og bruger 80 mia. kr. på at købe varer
  3. Butikkerne får flere penge og ansætter flere, som også bruger penge
  4. Den samlede effekt kan være 200-300 mia. kr. - mere end den oprindelige 100 mia.

7. Hvordan bruger vi dette i praksis?

Som finansøkonom eller Financial Controller møder du varemarkedet i din daglige arbejde:

  • Salgsprognoser: Du skal forstå, hvordan efterspørgslen påvirkes af forskellige faktorer
  • Prisstrategi: Du skal forstå sammenhængen mellem priser og efterspørgsel
  • Budgettering: Du skal tage højde for, hvordan økonomiske udsving påvirker virksomheden
  • Rådgivning: Du skal kunne forklare kunder, hvordan konjunkturer påvirker deres situation

8. Sammenfatning

Varemarkedet på kort sigt bestemmes af efterspørgsel og udbud. De vigtigste punkter er:

  1. Efterspørgslen afhænger af: Indkomst, priser, renter, forventninger og offentlige udgifter
  2. Udbuddet på kort sigt: Reagerer primært på priserne, da produktionskapaciteten er fast
  3. AD-AS modellen: Viser sammenhængen mellem samlet efterspørgsel og samlet udbud
  4. Ligevægt: Opstår, når efterspørgsel og udbud er lige store
  5. Økonomiske udsving: Opstår ved efterspørgsels- eller udbudschok
  6. Multiplikatoreffekten: Gør, at ændringer kan have større effekt end forventet
